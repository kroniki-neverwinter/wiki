# System języków

## Dostępne języki graczy

| Komenda | Język        |
|---------|--------------|
| !dr     | Druidzki     |
| !el     | Elfi         |
| !gn     | Gnomi        |
| !kr     | Krasnoludzki |
| !nb     | Niebiański   |
| !ni     | Niziołczy    |
| !or     | Orczy        |
| !ot     | Otchłanny    |
| !pi     | Piekielny    |
| !pw     | Podwspólny   |
| !sm     | Smoczy       |

## Stopnie znajomości języka

| Znajomość | Stopień             |
|-----------|---------------------|
| 0-30      | Brak                |
| 31-60     | Początkujący        |
| 61-80     | Średniozaawansowany |
| 81-94     | Zaawansowany        |
| 95-100    | Pełna biegłość      |

:::info

Języków można się uczyć w trakcie rozgrywki, wchodząc w interakcję z innymi graczami posługującymi się danym językiem.

:::
